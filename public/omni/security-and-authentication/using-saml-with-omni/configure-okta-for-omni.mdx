---
title: Configure Okta for Omni
description: Configure an Okta SAML application to authenticate users into your Omni instance.
---

1. Log in to Okta.
2. [Create a new app integration](https://developer.okta.com/docs/guides/create-an-app-integration/openidconnect/main/).
3. Select **SAML 2.0** as the sign-in method.
4. Give the application a clear, recognizable name (for example, Omni).
5. Configure the **SAML settings** and **Attribute Statements** as shown below:

<img src="./images/configure-okta-for-omni-okta-create-saml-app.png" alt=""/>

6. Click **Next**, optionally provide feedback, then select **Finish**.

Once setup is complete, go to the **Assignments** tab for the new application to manage user access as needed.
